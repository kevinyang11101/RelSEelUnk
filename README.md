# 大学生竞赛管理系统 java658

## 项目概述

大学生竞赛管理系统是基于SpringBoot和Vue构建的B/S架构系统，旨在为大学生提供便捷的竞赛信息管理、报名、作品打分等功能。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：IDEA、JDK1.8、Maven、MySQL
- 技术栈：Java、MySQL、SpringBoot、MyBatis、Vue

## 功能模块

### 系统角色

系统包含以下角色：

- 管理员
- 学生
- 教师

#### 管理员模块

- 教师信息管理
- 学生信息管理

#### 学生模块

- 竞赛信息管理
- 竞赛报名管理
- 作品打分管理

#### 教师模块

- 竞赛信息管理

## 运行环境

- 操作系统：Windows/Mac
- 数据库：MySQL
- 服务器：Tomcat

## 目录结构

```plaintext
project/
├── backend/        # 后端代码
├── frontend/       # 前端代码
└── document/       # 项目文档
```

## 部署步骤

1. 安装并配置MySQL数据库。
2. 部署后端代码到服务器，并配置相应的环境。
3. 部署前端代码到服务器或静态资源服务器。
4. 运行系统。

## 核心亮点

- 使用SpringBoot和Vue，前后端分离，易于维护和扩展。
- 严谨的权限控制，保证系统安全。
- 提供完整的功能模块，满足大学生竞赛管理的需求。
